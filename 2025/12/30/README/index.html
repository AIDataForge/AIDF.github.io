AI NAS - 企业级智能文档管理系统

项目简介

AI NAS (Artificial Intelligence Data Forge) 是一个基于 MAAS (Model as a Service) 的企业级智能文档管理系统，旨在为企业提供文档转换、知识锻造、智能检索、异常恢复与健康监控的完整能力栈。

核心特点

1. 从”文档存储”到”知识锻造”

不仅存储文档，还通过AI智能提取结构化知识，将非结构化文档锻造成统一、可信、可检索的知识底座。

2. 从”单一处理”到”多模态智能”

支持文档、图片、音频、视频等多种格式的智能处理和分析，提供全方位的文档理解能力。

3. 从”数据系统”到”智能平台”

提供完整的文档管理、智能对话、项目流程管理等企业级能力，构建智能化的知识工作平台。

技术栈

  • Web框架：FastAPI (异步Web框架)
  • 数据库：PostgreSQL (双数据库架构：ai_nas核心库 + ai_nas_app业务库)
  • ORM框架：SQLAlchemy
  • 文档处理：MarkItDown + 自研解析器
  • LLM集成：OpenAI API (支持多种LLM提供商)
  • VLM集成：视觉语言模型支持
  • 向量检索：Embedding + 向量数据库
  • 异步编程：Python asyncio
  • 任务调度：后台任务管理器
  • 版本管理：Alembic (数据库迁移)
