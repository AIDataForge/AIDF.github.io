Smart Chat 模块架构说明

1. 模块概述

Smart Chat 是AIDF的智能对话助手模块，提供通用智能对话、文档检索、问题分析等多种AI应用功能。

1.1 功能定位

  • 智能对话：基于LLM的自然语言对话
  • 文档检索：结合向量检索和全文检索的智能文档搜索
  • 问题分析：复杂问题的分解和分析
  • 知识解释：文档内容的理解和解释

1.2 访问路径

  • URL：/smart-chat
  • 文件位置：app/modules/smart_chat/static/smart_chat.html
  • 权限要求：正式用户（USER）或管理员（ADMIN）

3. 核心组件

3.1 前端组件

3.1.1 对话界面

功能：

  • 消息显示区域
  • 输入框和发送按钮
  • 执行器选择
  • 会话管理

实现：

  • 主应用类：SmartChatApp
  • 消息渲染：支持Markdown、代码高亮
  • 流式响应：支持实时流式输出

3.1.2 执行器选择

功能：

  • 显示可用执行器列表
  • 执行器描述和示例
  • 执行器选择切换

执行器类型：

  • 文档查询执行器
  • 数据分析执行器
  • 代码生成执行器
  • 任务分解执行器
  • 其他专业执行器

3.2 后端服务

3.2.1 Smart Chat服务

服务：app/modules/smart_chat/services/

功能：

  • 对话管理
  • 消息处理
  • 执行器协调
  • 上下文管理

3.2.2 执行器框架

功能：

  • 执行器注册和管理
  • 执行器调用和结果处理
  • 执行器链式调用

5. 技术实现

5.1 前端技术

  • HTML5：页面结构
  • CSS3：样式和动画
  • JavaScript (ES6+)：交互逻辑
  • Marked.js：Markdown渲染
  • Highlight.js：代码高亮
  • Stream API：流式响应处理

5.2 后端技术

  • FastAPI：API服务
  • 异步处理：对话处理异步化
  • LLM集成：OpenAI API
  • 向量检索：Embedding + 向量数据库

5.3 AI技术

  • 大语言模型：GPT系列模型
  • 向量嵌入：文本向量化
  • 检索增强生成：RAG技术
  • 上下文管理：会话上下文维护

6. 性能优化

6.1 前端优化

  • 流式渲染：实时显示生成内容
  • 消息缓存：会话消息缓存
  • 防抖处理：输入防抖

6.2 后端优化

  • 异步处理：对话处理异步化
  • 缓存机制：常见问题缓存
  • 批量处理：批量向量检索

7. 扩展性

7.1 执行器扩展

  • 支持自定义执行器
  • 支持执行器插件
  • 支持执行器链式调用

7.2 功能扩展

  • 支持更多对话模式
  • 支持多轮对话
  • 支持对话历史管理

8. 安全设计

8.1 访问控制

  • 需要用户认证
  • 会话隔离
  • 执行器权限控制

8.2 数据安全

  • 敏感信息过滤
  • 查询内容验证
  • 响应内容审核

9. 总结

Smart Chat模块是AIDF的智能对话核心，通过执行器框架和RAG技术，提供了强大的智能对话能力。模块设计灵活，支持多种执行器，具备良好的扩展性和安全性。
