文档中心模块API接口文档

1. 概述

文档中心模块使用的API接口，包括文件管理、文档搜索、文件上传等功能。

1.1 基础信息

  • Base URL：http://localhost:8000
  • API前缀：/api/v1
  • Content-Type：application/json（除上传接口外）

2. 文件管理API

2.1 获取文件列表

接口：GET /api/v1/files/

说明：获取文件列表，支持分页、排序、筛选

权限：需要用户认证

参数：

  • skip (int, 可选)：跳过的记录数，默认0
  • limit (int, 可选)：返回的记录数，默认20，最大100
  • directory (string, 可选)：目录过滤
  • search (string, 可选)：搜索关键词
  • status (string, 可选)：状态过滤（queued/processing/completed/failed/unanalyzed）
  • sort_by (string, 可选)：排序字段（created_at/file_size/title），默认created_at
  • sort_order (string, 可选)：排序顺序（asc/desc），默认desc

3. 文档搜索API

3.1 搜索文档

接口：POST /api/v1/chat2doc/search

说明：智能文档搜索，支持多种搜索模式

权限：需要用户认证

请求体：

参数说明：

参数 类型 说明
query string 搜索查询（必需）
mode string 搜索模式：fast/smart/vector/auto
output_format string 输出格式：standard/card/list/custom
user_id string 用户ID（可选，用于访问统计）
page int 页码（从1开始）
page_size int 每页大小（1-100）
similarity_threshold float 向量相似度阈值（仅vector模式）
allow_fallback bool 允许降级到传统搜索（仅vector模式）

5. 错误处理

5.1 常见错误

  • FILE_NOT_FOUND：文件不存在
  • FILE_UPLOAD_FAILED：文件上传失败
  • FILE_PROCESSING_FAILED：文件处理失败
  • PERMISSION_DENIED：权限不足
