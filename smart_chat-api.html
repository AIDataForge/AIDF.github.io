Smart Chat 模块API接口文档

1. 概述

Smart Chat模块使用的API接口，提供智能对话、执行器管理、会话管理等功能。

1.1 基础信息

  • Base URL：http://localhost:8000
  • API前缀：/api/v1/smart-chat
  • Content-Type：application/json

2. 对话API
